Bouncing Ball

You may have already used the coding language called Scratch at school. You can download the Scratch app from the Scratch web site so that you can use the editor without an internet connection or use it in the browser by just going here and clicking on 'Create'. You choose a 'sprite', (you can see a cat here) and then link together blocks to make one or more programs for it. Each sprite has its own set of programs. There are some 'getting started' videos here.

I decided to experiment with the animation of a bouncing ball. There are two sprites, a ball and a duck, and a little bit of maths to calculate how fast the ball is moving up and down. Try running the animation.

You might like to think of something you could do and create your own animation. See what you can create.
